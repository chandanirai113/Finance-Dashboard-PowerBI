# Finance Dataset Dashboard — Power BI

## Project Overview
An interactive Finance Dashboard built in Power BI analyzing 700+ rows of financial data across 5 segments, 5 countries, and 6 products.

## Key Insight
The Enterprise segment shows negative profit (-643K) despite being the most heavily discounted — indicating that aggressive discounting is hurting margins without driving proportional sales volume.

## Dataset
- Source: Microsoft Power BI Sample Dataset (Financial Data)
- 700+ rows, 16 columns
- Columns: Segment, Country, Product, Discount Band, Units Sold, Manufacturing Price, Sale Price, Gross Sales, Discounts, 
 Sales, COGS, Profit, Date, Month Number, Month Name, Year

## Tools & Techniques
- **Tool:** Microsoft Power BI Desktop
- **Data cleaning:** Power Query (data types, trim, null handling)
- **DAX Measures:** SUM, DIVIDE, conditional aggregations
- **Visuals:** KPI cards, bar chart, clustered column chart, discount analysis table

## Key Visuals Built
- KPI Cards — Total Sales (118.73M), Total Profit (16.89M), Profit Margin (14.23%)
- Bar chart — Profit by Segment (Government leads at 11.4M)
- Table — Country × Product × Segment profit breakdown
- Discount Band analysis — Loss vs Profit by discount level

## How to get the dataset
1. Open Power BI Desktop
2. Click Home → Try a sample dataset → Financial Sample
3. Power BI loads it automatically — no download needed
